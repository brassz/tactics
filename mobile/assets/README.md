# Assets

Coloque aqui os arquivos de imagem do app:

- `icon.png` - Ícone do app (1024x1024)
- `splash.png` - Splash screen (1242x2436)
- `adaptive-icon.png` - Ícone adaptativo Android (1024x1024)
- `favicon.png` - Favicon para web (48x48)

Para gerar ícones facilmente, você pode usar:
- https://www.appicon.co/
- https://icon.kitchen/

## Cores sugeridas

- Primária: #3B82F6 (Azul)
- Fundo: #FFFFFF (Branco)
